# Customer Churn Analysis

This project focuses on analyzing customer churn data to identify patterns and factors that contribute to customer loss. The project combines Python, MySQL, and Power BI to explore customer behavior and generate business insights related to customer retention.

## Tech Stack
- Python
- Pandas
- MySQL
- Power BI
- Jupyter Notebook
- VS Code

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- SQL query analysis
- Customer churn pattern analysis
- Interactive Power BI dashboards
- Business insight generation

## Dataset
- Telco Customer Churn Dataset

## Objective
The goal of this project is to better understand customer behavior and identify factors that may influence customer churn using data-driven analysis techniques.
